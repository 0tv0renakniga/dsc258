Evaluation Metrics




                     ● TP: True Positive
                        ○ Predicted as Positive & True Label Positive
                     ● TN: True Negative
                        ○ Predicted as Negative & True Label Negative

                     ● Similarly, we have …

                     ● FP: False Positive
                     ● FN: False Negative
Evaluation Metrics: Accuracy,
  Precision, Recall, and F1




                                ● Accuracy
                                  ○ (TP + TN) / (TP + FP + FN + TN)
                                ● Precision
                                  ○ TP / (TP + FP)
                                ● Recall
                                  ○ TP / (TP + FN)

                                ● F1
                                   ○ Harmonic Mean between Prec and Rec
                                   ○ 2 * Prec * Rec / (Prec + Rec)
